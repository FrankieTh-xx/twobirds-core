tb.bind() function

sample calls:

tb.bind( document.body )

  • scans the given element and all of its descendants in the DOM and looks for attributes "data-tb" in the nodes.
  • resulting list will be scanned for those nodes that do not already have an tb object inside.
  • creates a new tb object based on the class namespace given in the "data-tb" attribute
  • stores it in the DOM element

tb.bind( document.body, 'n1.n2.' [ , ] )

  • creates a new tb object based on the 2nd parameter, giving 3rd as constructor parameter
  • stores it in the DOM element THIS VARIANT WILL overwrite ANY tbo OBJECT THAT ALREADY RESIDES IN THE DOM NODE!

Parameters:

  • pSelector Object

    DOM node
